# Bezawada Hemanth Chowdary - Portfolio Website

A clean, minimalist, and responsive single-page portfolio website for an AI/ML Engineer and Data Scientist.

## Technologies Used
- HTML5
- CSS3 (Vanilla, Custom Properties)
- JavaScript (Vanilla)

## How to Open Locally
1. Clone or download this repository.
2. Open the `hemanth-portfolio` folder.
3. Simply double-click on `index.html` to open it in your default web browser. No server is required to view the site.
4. Alternatively, if using VS Code, you can use the "Live Server" extension for a better development experience.

## Deployment Instructions
Since this is a static website with no backend dependencies, it can be deployed easily on any static hosting provider.

**GitHub Pages:**
1. Create a new repository on GitHub.
2. Upload the contents of the `hemanth-portfolio` folder to the repository.
3. Go to the repository **Settings** -> **Pages**.
4. Select the `main` branch as the source and click Save.
5. Your site will be published at `https://<username>.github.io/<repo-name>`.

**Vercel / Netlify:**
1. Create a new project.
2. Drag and drop the `hemanth-portfolio` folder.
3. The platform will automatically deploy and provide a live URL.
